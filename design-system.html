Intro's left edge = logo's left edge. Always.

The panel is full-bleed (100vw) but its content sits in the same 1240px container as the rest of the page. Column rules are drawn in the gutter, never as a border on the column — that's what used to leave a stray line hanging off the last column.

05 Audit — What Was Broken
SeverityIssueCauseFix
HIGH Mega menu had no containing block .mega is position:absolute, but neither .header nor .navitem was positioned — so it resolved against the page. Added position:relative to .header-in.
HIGH Intro block off the page grid Intro started 58px inboard of the container; it aligned to nothing on the page. Rebuilt grid so intro's left edge = logo's left edge (verified 0px delta, 1120–1920px).
MED Stray rule off the last column border-right on every .mega-col drew a line into empty gutter. Separators moved into the gutter via ::after; content now fills its track.
MED Green top rule clipped Header's 2px bottom border sat over the panel; top:100% landed above it. margin-top:2px — panel now flush to header (verified 0px).
MED Header overflowed at 1041–1180px Logo + 5 nav items + actions + gutters = 1077px inside a 1060px viewport. Compact band: secondary links drop, nav padding tightens. Zero overflow at all widths.
LOW Uneven column rhythm gap:0 + per-column padding + intro margin gave 24/21.6/0px spacing. Single --mega-gap token (32px) drives all four tracks.
LOW Dead + duplicated CSS A 980px 2-col mega rule could never fire (drawer takes over at 1040px); the mobile block was pasted twice. Unreachable rules removed; mobile reset applied to both blocks.
06 Do's & Don'ts

✓ Do

  • Anchor to the page grid.If a panel is full-bleed, its content still lands in the 1240px container.
  • Draw rules in the gutter.Separators belong between columns — not as a border that eats content width.
  • Let green carry hierarchy.A 4px underline or left-border says "active" more clearly than a fill.
  • Keep the 38px row.Every mega link is the same height, so scanning is a straight line down.
  • Use one token per decision.--mega-gap once beats 32px typed four times.

✕ Don't

  • Don't absolutely position without a parent.It'll look fine at your width and drift at every other one.
  • Don't stack accents.Orange, lime, and cyan each have one job. Two at once reads as a warning.
  • Don't round the corners.2px UI / 4px panel. Anything softer stops looking industrial.
  • Don't leave dead breakpoints.A rule below the drawer threshold never fires — it just misleads the next person.
  • Don't let the header overflow.Shed secondary links before you let the page scroll sideways.
